### Installing

```
git clone git@github.com:serebro/p81-blocks.git
cd p81-blocks
```

### Running
```
docker-compose up -d
```

### Displaying results
```
docker logs app
```

Example of "Block Actual Workload":
```
[["triangle A1","USED"],["triangle A2","USED"],["triangle A3","UNUSED"],["square B1","USED"],["circle C1","USED"],["circle",null],["rhomb E1","USED"]]
```
