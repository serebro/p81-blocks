### Installing

```
git clone git@github.com:serebro/p81-blocks.git
```

### Running
```
docker-compose up -d
```

### Display results
```
docker logs app
```

Example of "Block Actual Workload":
```
["triangle A1","triangle A2","triangle A3","square B1","circle C1","circle","rhomb E1"]
["USED","USED","UNUSED","USED","USED",null,"USED"]
```
